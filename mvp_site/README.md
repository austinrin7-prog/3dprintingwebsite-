# PrintForge Studio MVP

This is a static MVP storefront for selling ready-made and custom 3D printed products.

## Open It

Open `index.html` in a browser.

## Included

- Homepage with store positioning and product visuals
- Product catalog with category filters
- Add-to-cart drawer with local storage
- Demo checkout button
- Custom print request form with seller-ready summary
- Simple seller production snapshot
- Seller inventory manager for adding and removing shop items
